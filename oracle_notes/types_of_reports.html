Types of Reports

In Oracle Apex we mainly have four types of Reports.

Interactive Grid : IG allows users to view & edit data direclty in the tabular formats.

Interactive Reports : IR is mainly for data analysis with filtering and customization options but no direct editing

Classic Report : Is a simple SQL-based reports with unlimited customization.

Report and Form is a combination where users select a record from a report and edit it in a seperate form pages.
