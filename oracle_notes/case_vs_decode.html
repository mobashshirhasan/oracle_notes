Decode

Decode is an oracle-specific function used to compare one expression with multiple values.

It works only for equality conditions and returns a corresponding result.

Ex:- select decode(:a, 'APPLE', 1, 'apple', 2) d from dual.

Case

A case statement is used to apply conditional logic in SQL.

It works like IF-ELSE in the programming language and return a value based on condition.

Ex:- select case when :a = 'Apple' then 1 when :a = 'apple' then 2 else 3 end d from dual;
