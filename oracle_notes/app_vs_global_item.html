Application Item.

Application Item is a Global Variable for the whole application.

Application Item have not associated with a page.

We cannot get the value of the application item using JavaScript directly.

Application Item created under the shared components then application item

Can be accessible from any pages, any process, and any PL/SQL block.

Application Item store values for the entire user session.

Global Page Item

Global Page is a page item created on page 0.

Global page Item appears on all pages.

Global page item have an interface in the form of Rectangular shape.

Global page Item can be a text field, text area, password, select list, check box, and so on.

We can get value of the Global Page Item using JavaScript.

We can control the Global Page Item using Server Side Condition.
